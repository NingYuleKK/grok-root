# 礼物探险季-宝石大冒险

语音直播间礼物玩法 H5 可交互原型。

## 运行

```bash
npm run dev
```

打开 `http://127.0.0.1:5188`。

## 当前玩法映射

- Tab 1：获得锄头
  - 收/送任意 1 个铜锄主题礼物，获得铜锄 x1
  - 收/送任意 1 个银锄主题礼物，获得银锄 x1
  - 收/送任意 1 个金锄主题礼物，获得金锄 x1
  - 送出帝王套或完成每轮大满贯，获得金刚锄 x1
- Tab 2：挖矿纪念品
  - 铜锄挖出水晶
  - 银锄挖出珍珠
  - 金锄挖出黄金
  - 金刚锄挖出钻石
- 兑换
  - 水晶兑换发言气泡
  - 珍珠兑换小座驾
  - 黄金兑换大座驾
  - 钻石兑换对话 CP 座驾
  - 水晶/珍珠/黄金/钻石可组合兑换首饰礼物 A/B/C

## 下一轮可调参数

- 活动时间展示是否要写成具体日期，而不是 `3-5`
- 主题礼物是否需要具体礼物名和图片
- 每轮大满贯是单个用户完成，还是全房间累计完成
- 兑换奖励是否限量、是否有排行榜、是否区分送礼方/收礼方库存
