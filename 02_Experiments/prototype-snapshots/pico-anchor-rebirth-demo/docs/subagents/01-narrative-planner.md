# Subagent 1: Narrative Planner

## Scope

为《主播大作战：重生之我在 PICO 当主播》生成 9 个重生章节的叙事配置，供章节入口、礼物墙、证书、头像框、分享卡、飘屏和个人页档案消费。

## Outputs

- `src/data/chapters.ts`
- `src/data/demoUsers.ts`

## Rules

- 每章固定 `wallSize = 12`。
- 每章里程碑固定为 `4 / 8 / 12`。
- 每章礼物固定 12 个，`low / mid / high` 各 4 个。
- 每章有 1 个帝王套和 3 到 5 个伴手礼。
- 默认演示章节为 `玫瑰沙龙会客厅主人`。

## Rose Salon Notes

玫瑰沙龙章节已保留指定礼物：玫瑰请柬、小折扇、羽毛笔、糖霜点心、蕾丝手套、珍珠手账、茶会银匙、贝母麦克风、玫瑰马车、水晶吊灯、祖母绿冠冕、鎏金画像。

伴手礼来源说明统一为：“由主播 桃桃 于《玫瑰沙龙会客厅主人》通关后亲赠。”
