# 房间龙池 Demo v0.1

单页 React demo，用来验证语音房里“大家一起把龙池点亮”的同时在线氛围。

## 运行

```bash
npm install
npm run dev
```

当前核心验证点：

- 点击礼物按钮后，龙气上涨，公屏出现贡献记录。
- 进度到 70%、90% 后，文案和动效进入临界状态。
- 龙气达到 100 后，触发一次房间显影事件，并保留溢出龙气进入下一轮。
- 显影只随机表现形式，不随机经济奖励，避免抽奖/博彩感。

## 检查
